What is WeeB?

WeeB (proper name WeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeBasic) is an esoteric programming language invented by None1, you can see the original specification at the Esolang wiki.
WeeB is a derivative of the esoteric programming language Weeeeeeeeeeeeeeeeeeeeeeeeeeeeee. It adds BASIC-like syntax, including if statements, while loops and string printing, making it high level. And it removes some bad commands in Weeeeeeeeeeeeeeeeeeeeeeeeeeeeee, such as the useless forward jump command and the program start and termination commands.

A later Turing-complete derivative WeeB++ (a.k.a WeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeBasic++), expands memory
to a tape and a pointer, and adds some extra commands and some changes on the original commands.

External resources

WBC is a compiler from WeeB/WeeB++ to C.
